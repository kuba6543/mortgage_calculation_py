# Wykonywane zadanie: nr 1

## Description of programme:

Program will count and show mortgage schedule in table.

## Variables in programme:

* amount_of_mortgage
* bank_fee - in per cent
* interest_rate - in per cent, per year
* period_of_payment - period of payment in months

## Functions in programme:

* read_data() - reading data from user
* calculations(amount_of_mortgage, bank_fee, interest_rate, period_of_payment) - function calculating installment
* make_a_table() - makes a table out of data
